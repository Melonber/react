Команды в терминале для запуска:

npm i -g expo-cli

expo init нзвание_проекта

cd нзвание_проекта

npm i

npm start

После скана qr-кода запускается приложение, в качестве функционала есть кнопка, которая при нажатии увеличивает цифру на экране:
